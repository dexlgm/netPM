# NetPM
The NET Packet Manager repo for Zenith!

## Net Paket Manager
In ZenithOS You can install binaries from this repo, from gitpages site^^

## How to use it?
It's easy! You can download binaries just for one command
```zsh
netinstall <package name>
```
It downloads files directly to /bin (PATH), you can start it after install
For download binary to other directory use:
```zsh
netinstall <pkg> -o <output dir>
```
This command download file INTO o. dir!
